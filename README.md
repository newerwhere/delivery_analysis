# Delivery Analysis Test Task

Аналитическая сводка по данным заказов за 7 дней.

## Структура анализа

1. Первичный анализ данных
2. Общая статистика заказов
3. Анализ по часам заказа
4. Анализ по складам
5. Детализация: склад × час
6. Анализ нагрузки на курьеров

## Основные выводы

- доля просрочек ≈ 11.5%
- основная часть задержек возникает на этапе доставки
- наблюдается рост просрочек в часы пик
- отдельные склады демонстрируют повышенную долю просрочек
- на некоторых складах наблюдается высокая нагрузка на курьеров

## Используемые инструменты

Python, Pandas, SQL, Matplotlib, Seaborn
